# News React App

Mobile-first React app (News) with json-server backend sample data.

## Run locally

1. Install dependencies:

```powershell
npm install
```

2. Start json-server and dev server together:

```powershell
npm run dev
```

The app runs at `http://localhost:3000` and json-server at `http://localhost:5002`.

## What to push

- `src/`, `public/`, `package.json`, `db.json`, and assets.
- `.gitignore` already present to exclude `node_modules` and build artifacts.
